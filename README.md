# 🌤️ SkyVolt

**Know the sky. Power your day.**

SkyVolt is a modern **Weather & Solar Intelligence Dashboard** that combines real-time weather information with solar energy estimation. It helps users understand current weather conditions, estimate daily solar power generation, calculate electricity savings, and make smarter energy decisions.

## 🔗 Live Demo

👉 [https://skyvolt-weather-solar-dashboard.vercel.app](https://skyvolt-weather-solar-dashboard.vercel.app)

---

## ✨ Features

* 🌦️ Real-time weather information
* 🌡️ Current temperature and feels-like temperature
* 💧 Humidity
* 🌬️ Wind speed
* 📊 Atmospheric pressure
* ☀️ UV Index
* 👀 Visibility
* 💦 Dew Point
* 🌧️ Rain Probability
* 🌅 Sunrise & Sunset
* 🌙 Moon Phase
* 🌫️ Air Quality Index (AQI)
* ⏰ Hourly Weather Forecast
* 📅 5-Day Weather Forecast
* ⚠️ Severe Weather Alerts
* ☀️ Solar Energy Estimator
* ⚡ Estimated Solar Output
* 💰 Estimated Electricity Savings
* 🧺 Laundry Drying Recommendation
* ❄️ AC Runtime Estimation
* 💡 Smart Energy Saving Tips
* ⭐ Favorite Cities
* 🔍 City Search
* 📱 Responsive Design

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/saher-eman/skyvolt-weather-solar-dashboard.git
```

### Install dependencies

```bash
npm install
```

### Start the server

```bash
npm start
```

Open your browser and visit:

```
http://localhost:3000
```
